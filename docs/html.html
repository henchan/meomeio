html.js

¶

html.js holds synchronous html processing logic for meo.my adaptors.
The adaptor.html() function is called asynchronously from the Chrome context page and executed in Chrome's background process. It gets called by any active adaptor whose spec includes at least one active html item.
The function of this code is to extract meta data from a given page of HTML and return an array of tag strings

			return unsanitised.replace(/[!"#$%&'()*+,.\/:;<=>?@[\]^`{|}~]/g, function(c) {
				return escChars [c];
